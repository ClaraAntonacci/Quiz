# Quiz-Matemática 🧮  

## 📌 Introdução  

O **Quiz-Matemática** é um aplicativo desenvolvido com o objetivo de reforçar o aprendizado dos alunos de forma interativa e dinâmica. O projeto foi criado como atividade prática, permitindo aplicar conhecimentos de lógica, programação e desenvolvimento de aplicativos, além de estimular o raciocínio matemático por meio de perguntas e respostas.
O quiz aborda conteúdos matemáticos trabalhados em sala de aula, promovendo revisão e fixação do conteúdo de maneira mais divertida e acessível.
